# 🎭 Playwright Test Automation Framework (TypeScript)

## 🚀 Tech Stack
- 🎭 **Playwright** – End-to-end testing framework for web apps
- 🟦 **TypeScript** – Strongly-typed JS for scalable tests
- 📦 **Node.js** – Runtime environment for executing tests
- 🧪 **Test Runner (Playwright Test)** – Test orchestration and assertions
- 📊 **HTML Reporting** – Beautiful test reports for analysis
